# ASCII
American Standard Code for Information Interchange   
: 미국 정보 교환 표준 코드    
: 7비트(128개)로 구성된 코드       


**구성**   
- 제어 문자 33개
- 문자 95개
    - 1개 - 공백
    - 10개 - 숫자
    - 32개 - 특수 문자
    - 52개 - 영문 대소문자


ASCII 코드 | 문자
---|---
33  | !
34  | "
35  | #
... | ...


**ASCII 확장**   
- ISO/IEC646 -- 7비트
- ISO 8895 -- 8비트
