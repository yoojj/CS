# Type System
: 데이터에 타입을 할당하여 타입을 기준으로 데이터 분류           
: 사용자에게 데이터에 대한 추가적인 추상적 정보 제공     
: 프로그램 내부에서 데이터 타입을 통해 데이터를 정확하게 사용 ex. 1 != '1'
: 타입 체크에 따라 컴파일 시점에 데이터 타입을 검사해 오류 감지     


**+ type**  
: 어떤 값을 다른 값과 분별할 수 있는 고유한 특성


**분류**
- 데이터 타입  
    - 기본 자료형
    - 구조적 자료형
    - 포인터 자료형
- 타입 안전
    - strong type
    - weak type
- 타입 체크
    - static type
    - dynamic type
    - optional type
