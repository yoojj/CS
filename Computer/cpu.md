# CPU
Central Processing Unit     
: 중앙처리장치     
: 컴퓨터를 구성하는 핵심 구성 요소    
: 컴퓨터 시스템 전체를 작동하고 통제하고 연산을 수행하는 물리적 장치     
: 초기에는 진공관으로 구성되었으며 현재는 수십억 개의 트랜지스터로 구성된 칩    


**진공관**  
: 전기를 한 방향으로 흐르게 통제하는 부품으로 전기 흐름 여부를 구분해 신호로 사용     


\+ MCU (Micro Controller Unit)   
\+ MPU (Micro Processor Unit)  


**구성 요소**     
- 제어 장치
- 연산 장치
- 기억 장치
